# leitura-espacada

Transforma leitura em aprendizado retido. Baseado no algoritmo SM-2.

## Quando usar

- Estudar qualquer conteúdo com retenção de longo prazo
- Criar cards de estudo de artigos, livros, pesquisas
- Implementar revisões espaçadas em leituras diárias
- Organizar conhecimento por temas de forma persistente

## Fluxo típico

```
Artigo/conteúdo → criar cards → agendar revisões → revisar conforme SM-2
```

## Integração com pesquisa

Ideal como etapa final após `perplexity`, `deep-research` ou `youtube-transcript`:

```
Pesquisa → sintetizar → leitura-espacada (criar cards) → revisar
```
