# Grok (xAI)

Dois modos de uso: busca e modelo.

## grok-search (busca)

Busca estruturada em X/Twitter e web via ferramentas `web_search` e `x_search`.

```
triggers: grok, search x, search twitter, find tweets, x search
```

**Melhor para:**
- Tweets, threads e usuários do X
- Alternativa estruturada ao Brave (retorna JSON + citações)
- Monitorar discussões sobre um tema no X

## xai (modelo)

Acesso aos modelos Grok completos para análise e raciocínio.

- **Grok-3** — raciocínio profundo, análise longa
- **Grok-3-mini** — rápido, tarefas simples
- **Visão** — análise de imagens

**Quando preferir Grok sobre Claude:**
- Perspectiva diferente em decisões estratégicas
- Raciocínio com contexto de X/Twitter
- Deep reasoning tasks longas
