# Clawdbot Skills Index

Ponto de entrada do skill graph. Cada cluster abaixo é um MOC (Map of Content) com skills interconectadas.
Siga apenas os links relevantes para a tarefa atual — não leia tudo.

## Clusters

### 🎯 Marketing & Conteúdo
Skills para criação de conteúdo, estratégia de redes sociais e análise de engajamento.
→ [[marketing/MOC]] — copywriting, storytelling, calendário editorial, análise de engajamento

### 🤖 Agent & Workspace
Skills para manutenção, melhoria e governança de agentes e workspaces.
→ [[agents/MOC]] — manutenção, memória, self-improvement, avaliação

### 🔍 Pesquisa & Conhecimento
Skills para busca, síntese e gestão do conhecimento.
→ [[research/MOC]] — perplexity, deep-research, grok, context7, leitura-espaçada

### 📋 Produtividade & Projetos
Skills para PRDs, tarefas, reuniões e fluxos de trabalho.
→ [[productivity/MOC]] — prd, reunião, tarefas, brainstorming, council

### 🛠️ Dev & Automação
Skills para código, browser, GitHub e automação técnica.
→ [[dev/MOC]] — coding-agent, playwright, github, tmux, parallel-task

### 🔐 Segurança
Skills para proteção do agente e monitoramento.
→ [[security/MOC]] — prompt-guard, clawsec-suite, daily-security-triage

### 📦 Skills Management
Skills para criar, avaliar e distribuir outras skills.
→ [[meta/MOC]] — skill-creator, skill-evaluator, skills-manager, clawhub

### 🔗 Integrações Externas
Skills para APIs e serviços externos.
→ [[integrations/MOC]] — notion, obsidian, email, twitter, youtube, 1password
