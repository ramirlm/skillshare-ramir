# reuniao-para-prd

Converte qualquer transcrição de reunião em entregáveis estruturados.

## O que gera

1. **Insights práticos** — decisões tomadas, pontos de ação
2. **Desafios e pontos cegos** — o que pode estar faltando
3. **PRD opcional** — documento de requisitos se houver solução/implementação

## Como usar

Colar transcrição no chat ou fornecer arquivo → skill processa automaticamente.

## Quando preferir vs. ontologia-transcript

| | reuniao-para-prd | ontologia-transcript |
|--|-----------------|---------------------|
| Objetivo | Ação/PRD | Preservar conhecimento |
| Output | Insights + PRD | Resumo + ontologia + vault |
| Melhor para | Reunião de planejamento | Reunião rica em decisões/contexto |

Usar os dois em sequência para cobertura completa.
