# self-improvement

Fecha o loop de aprendizado: cada erro vira conhecimento estruturado.

## Quando acionar

- Comando ou operação falha inesperadamente
- Ramir corrige o agente
- Ramir pede capacidade que não existe
- API ou ferramenta externa falha
- Agente percebe que seu conhecimento está desatualizado
- Abordagem melhor é descoberta para tarefa recorrente

## O que captura

```
erro/correção → categoria → root cause → solução → persistir
```

Persiste em `memory/` (operacional) ou `~/Obsidian` (aprendizado durável).
