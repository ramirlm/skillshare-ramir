# Storytelling

Narrativa transforma informação em experiência. Usar quando o objetivo é criar conexão, não só informar.

## Estrutura narrativa clássica

```
GANCHO — situação inicial ou tensão
VIRADA — o momento que mudou tudo
LIÇÃO — o que isso significa pro leitor
CTA — o próximo passo
```

## Arcos para marketing de conteúdo

### Antes/Depois
> "Eu achava que precisava postar todo dia. Hoje posto 3x por semana e cresci 40%."

### Jornada com obstáculo
> Setup → conflito → tentativa → fracasso → aprendizado → solução

### O especialista que errou
Admitir um erro passado cria autoridade + confiança ao mesmo tempo.

### Terceiros (case/cliente)
Contar a história de alguém que passou pelo problema cria identificação sem autopromoção.

## Elementos que amplificam

- **Detalhe sensorial**: um número concreto, uma data, um lugar específico
- **Diálogo interno**: o que você pensou/sentiu, não só o que fez
- **Tensão sustentada**: não resolver cedo demais
